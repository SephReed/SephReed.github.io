Hey! This project is still very much in development, so there are a few nuances:
  • Collision detection is still pretty wonky if you're moving too fast. Taping jump while falling is a good way not to fall through the world
  • The water dissapears when you're below it
  • Shadow mapping is very limited (and only appears in your immediate area)
Hope you have fun! -Seph

Controls:
  • WASD = Move
  • ARROWKEYS/MOUSE = Look around
  • SPACE = Jump / Fly Up
  • DOUBLE TAP SPACE = Fly On/Off
  • SHIFT = Run
  • CTRL = Fly Down / Crouch
  • R = Interact (colorful things tend to be interactive)
  • Q/E = Roll
  • I = Initialize
