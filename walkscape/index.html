Hey! This project is still very much in development, so there are a few nuances:
  • Collision detection is still pretty wonky if you're moving too fast. Taping jump while falling is a good way not to fall through the world
  • The water does dissapears when you're below it
  • Shadow mapping is very limited (only appears in your immediate area
This is a personal project in a very early stage, and pretty much nobody sees it but me, so I've only built it to work for me up to this point. Sorry. -Seph

Controls:
  • WASD/GDST = Move
  • ARROWKEYS/MOUSE = Look around
  • SPACE = Jump / Fly Up; DOUBLE JUMP = Fly On/Off
  • SHIFT = Run
  • CTRL = Fly Down / Crouch
